My Computer is Running Slow

December 9, 2016
Jamie Harvey

"My computer is running slow.  Can you come look at it?"  These words are the bane of my existence.  They get uttered on a weekly basis.  Although time consuming, this problem is generally an easy fix. 

For a Windows 7 or 10 computer (most likely in an Enterprise environment) hit CTRL + SHIFT + ESC.  This will bring up the Task Manager.  Check the Task Manager for processes that are taking up a majority of CPU, Memory, or Disk.  If any of these processes is being a resource hog, consider ending the process (don't kill anything vital).  Chances are, there's an application open (like Skype) that isn't being used and has no business being open.  Again, don't be an idiot and end a system process.  Most of the time "extra" programs get installed when a user downloads a program like Adobe Reader DC.  Ideally your system should be locked down with permissions and people using standard, non-administrator accounts (more about that on a different blog post) and users shouldn't be installing things like Adobe on their own in a corporate environment.  So if you notice 2 or more anti-virus systems "fighting it out" on the computer and using system resources, you'll need to get rid of a program or two.

Once you find these programs/processes, there are two responses.  You can disable, or delete.  If it's a program that you actually use sometimes, but want to disable from starting automatically, and you're on Windows 10, then in the Task Manager go to the Startup tab.  Select this (and any other) programs from the list and click disable.  This will prevent them from starting when you boot your computer (a major source of slow boot times).  On Windows 7 hit WIN + R and type msconfig and hit enter.  Then go to the Startup tab and disable these programs.  If you want to delete then go to Programs and Features in the Control Panel and uninstall the programs.

If it's still slow consider adding more RAM, upgrading the hard disk to solid state, or otherwise upgrading the system.
