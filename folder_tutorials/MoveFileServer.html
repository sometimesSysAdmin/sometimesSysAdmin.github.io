Moving A File Server

To CentOS 7


< Back to Tutorials

At work one of the file servers had begun to die frequently.  We recently upgraded a few other file servers, so we used one of the old(ish) servers as our replacement.  I entered the RAID configuration and created a RAID 5 array.  Then I installed CentOS 7.  I transferred the following files from the file server that was to be replaced:
  • /etc/exports (I copied this file directly to /etc/exports in the new server)
  • /etc/group
  • /etc/passwd
I used the group and passwd files in conjunction with useradd and groupadd to create users and groups with the correct uid's and gid's.
