Zeroshell

I have a very old computer and didn't want to waste what little computing power it offered.  While searching for a project to use it for, I came across Zeroshell.  This is a Linux based OS that can be used for various Network Administration tasks.  I have always seen the 802.1X option on my router, and never payed it much attention.  Realizing that I can use this old computer for WPA Enterprise encryption seemed like the perfect project for this computer that is long past its EOL.

I used Rufus to create a bootable flash drive using the DD image option and the Zeroshell-3.1.0-USB.img.gz image.
