Setting up a Splunk Server

< Back to Tutorials


We are going to set up a CentOS 7 server with Splunk now.  The documentation at splunk is great for installation so use that.  Install a Splunk server and download the Windows Universal Forwarder on dc2. 

In a browser, go to splunk.harvey.virtnet:8000.  The default credentials of "admin" and "changeme" should be changed (for obvious reasons).  Make sure that the Application Forwarder shows up in the browser at System -> Forwarder Management
